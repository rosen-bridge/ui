## Introduction

`@rosen-bridge/ui` is a simple ui kit containing react components that are used
inside other Rosen bridge projects.

## Installation

npm:

```sh
npm i @rosen-bridge/ui
```

yarn:

```sh
yarn add @rosen-bridge/ui
```
